HTML 語法完整教學

HTML（超文本標記語言）是創建網頁的標準標記語言。HTML描述了網頁的結構，由一系列的元素組成，這些元素告訴瀏覽器如何展示內容。

本教學將詳細介紹HTML的各種標籤和語法，幫助您從零開始學習網頁開發。

1. HTML 基礎結構

  • <!DOCTYPE html> - 定義文件類型
  • <html> - 包含整個 HTML 文件
  • <head> - 包含元數據（如標題、字符集）
  • <body> - 包含可見的頁面內容
